# Domain Vocabulary

Domain terms for Business Network + Catalogs in Retail/Brands/CPG.

## Core Terms

| Term | Definition |
|------|------------|
| **Business Network** | The platform/network connecting trading partners |
| **Catalog** | Collection of SKUs/items with their attributes |
| **SKU** | Stock Keeping Unit - unique product identifier |
| **Item** | A product record in the catalog |
| **Attributes** | Properties of an item (name, price, dimensions, images, etc.) |
| **Hierarchies** | Category trees (dept > class > subclass > item) |
| **GTIN** | Global Trade Item Number - barcode identifier |

## Systems

| Term | Definition |
|------|------------|
| **PIM** | Product Information Management system |
| **MDM** | Master Data Management |

| Term | Definition |
